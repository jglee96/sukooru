# @sukooru/svelte

Svelte adapter for Sukooru scroll restoration.

## Install

```bash
pnpm add @sukooru/svelte
```

## Usage

```svelte
<script lang="ts">
  import { createSukooruProvider } from '@sukooru/svelte'
</script>
```

Full documentation and examples:

- https://github.com/jglee96/sukooru#readme
